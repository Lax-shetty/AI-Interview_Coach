# AI Interview Coach 🎯

An AI-powered interview preparation tool that generates tailored interview questions and answer frameworks based on a job description and your resume.

## What it does

- Paste any job description and your resume/background
- Choose your interview focus (Behavioral, Technical, Product Thinking, Leadership, or Balanced)
- Get 10 tailored interview questions instantly
- Click any question to reveal an ideal answer framework specific to your profile and the role

## Built with

- Vanilla HTML, CSS, JavaScript
- Claude API (claude-sonnet-4) by Anthropic

## How to run

1. Clone the repo
2. Open `index.html` in your browser
3. Enter your [Anthropic API key](https://console.anthropic.com)
4. Paste a job description and your background
5. Hit Generate
